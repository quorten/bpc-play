Detailed Description

Functions for the wave editors.

Not only does this module contain functions for the wave editors, it also contains user interface helper functions and functions that prepare for file handling. However, some of the file handling code was moved to file_business.h to solve problems with different Microsoft runtime versions when this program is compiled with Microsoft Visual Studio.
