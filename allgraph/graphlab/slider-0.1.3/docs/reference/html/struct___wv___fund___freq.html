float _Wv_Fund_Freq::phase_pos

Current phase position in audio playback.

Ranges from 0.0 to 1.0, with 0.0 indicated playback is at the beginning of the fundamental frequency's sine wave and 1.0 indicating playback is at the end of one period respectively. Phase positions of different fundamentals will get off from their expected positions when the user changes the frequency of fundamentals in real time, which is why each fundamental's phase position must be tracked separately.

Wv_Editor_Data_Ptr_array* _Wv_Fund_Freq::wv_editors

Used for tracking of editor windows.

This field is an array of pointers rather than an array of structures so that the addresses of each structure will not change when the array changes. This is important because the signal handlers need a constant address for their user_data.
